# CS 307: Computer Graphics Assignments

This repository contains all the assignments from Wellesley's 7-week Computer Graphics course (CS 307) in Term 3 of Spring 2021. The assignments consist of web-based computer graphics displays using software that integrates JavaScript, WebGL, and Three.js (3D library). Feel free to interact with the assignments [here](https://peyton-a-wang.github.io/cs307-wellesley-cg/). 

- [Assignment 1: Obelisk](https://github.com/peyton-a-wang/cs307-wellesley-cg/tree/master/hw1)
  - Introduction to building objects out of raw geometries in Three.js.
- [Assignment 2: Clown](https://github.com/peyton-a-wang/cs307-wellesley-cg/tree/master/hw2)
  - Using embedded frames to build objects out of objects.
- [Assignment 3: Moving Cameras and Lighting Scenes](https://github.com/peyton-a-wang/cs307-wellesley-cg/tree/master/hw3)
  - Setting up cameras and experimenting with different lights.
